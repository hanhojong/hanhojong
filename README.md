* 주제선정 배경
  * 점점 고령화 인구가 증가하면서 노인복지시설의 부족함으로 인한 문제가 대두되고 있어 노인복지시설이 가장 시급한 지역을 알기 위해 분석

  3. 데이터 분석: 히트맵을 통해 행정동별 데이터 시각화 후 군집 분석을 통해 최적입지선정
  4. 결론 및 기대효과: 노인복지시설 최적입지로 9개의 행정동을 선정
